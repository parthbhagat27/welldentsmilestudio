# Image Library — WellDent Smile Studio

Drop your high-resolution photos into the folders below. Filenames must match exactly so the site picks them up automatically. Until you upload a photo, the site shows an elegant branded gradient placeholder — never a blurry image.

## Recommended specs
- Format: `.jpg` (photos) or `.webp` (preferred for web)
- Resolution: at least **1600px on the long edge** (2400px for hero shots)
- Compression: 75–85% quality
- Aspect ratio: don't worry — the site crops responsively (`object-cover`)
- Color: natural, bright, warm clinic light
